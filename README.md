## Как запустить проекты

### Вариант 1. Выгрузка базы (.dt)
1. Создай новую информационную базу.
2. В конфигураторе выбери «Администрирование → Загрузить информационную базу».
3. Укажи файл `dt/DemoBase_Portfolio.dt`.
4. Запусти 1С в режиме предприятия.

### Вариант 2. Внешние обработки (.epf)
1. Открой 1С в режиме предприятия.
2. Выбери «Файл → Открыть».
3. Выбери нужную обработку в папке `epf/`.
